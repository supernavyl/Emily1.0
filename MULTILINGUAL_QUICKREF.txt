╭──────────────────────────────────────────────────────────────────────╮
│  ⚙️ HOW TO USE                                                       │
╰──────────────────────────────────────────────────────────────────────╯

  AUTOMATIC LANGUAGE DETECTION:
    • Just type or speak in your language
    • Emily automatically detects and responds in that language
    • No configuration needed!

  CONFIGURE STT LANGUAGE (optional):
    Edit config.yaml:

    stt:
      language: "es"    # Spanish
      language: "fr"    # French
      language: "de"    # German
      language: null    # Auto-detect (recommended)

  START EMILY:
    ./scripts/start-emily.sh gui

╭──────────────────────────────────────────────────────────────────────╮
│  🔧 TECHNICAL SPECS                                                  │
╰──────────────────────────────────────────────────────────────────────╯

  WHY QWEN3 FOR MULTILINGUAL:
    • Trained on 36 TRILLION tokens
    • 128K context window (14B/32B models)
    • Native multilingual tokenization
    • Better non-English performance than Western models
    • Strong code-mixing support

  BACKEND:
    • TabbyAPI (ExLlamaV2) for text generation
    • Ollama for vision and embeddings
    • All running locally on your hardware
    • Zero cloud dependency

  VRAM USAGE:
    • Qwen3-14B:  ~10 GB  (most conversations)
    • QwQ-32B:    ~20 GB  (complex reasoning)
    • MiniCPM-V:  ~8 GB   (vision, on-demand)
