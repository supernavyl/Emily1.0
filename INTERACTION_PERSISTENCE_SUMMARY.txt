┃  HOW IT WORKS                                                           ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

  Every conversation turn:

  User types/speaks → Saved IMMEDIATELY to interactions.db
                      ↓
                   Metadata: timestamp, session, confidence

  Emily responds   → Saved IMMEDIATELY to interactions.db
                      ↓
                   Metadata: model, critic score, latency

  📁 Storage: data/interactions.db (SQLite with full durability)
  💾 Backups: data/backups/ (every 30 minutes)

╭──────────────────────────────────────────────────────────────────────╮
│  🔒 DATA GUARANTEES                                                  │
╰──────────────────────────────────────────────────────────────────────╯

  ✅ Write-through persistence (every turn saved before continuing)
  ✅ Crash-safe (even if Emily crashes, no data lost)
  ✅ Power-safe (full fsync to disk)
  ✅ Redundancy (separate from episodes.db)
  ✅ Searchable (full-text search index)
  ✅ Exportable (JSON for external use)

╭──────────────────────────────────────────────────────────────────────╮
│  📊 WHAT GETS SAVED                                                  │
╰──────────────────────────────────────────────────────────────────────╯

  For EVERY user turn:
    • Unique ID (UUID)
    • Session ID
    • Exact timestamp
    • Full text content
    • Importance score
    • Metadata (STT confidence, language, etc.)

  For EVERY Emily response:
    • Unique ID (UUID)
    • Session ID
    • Exact timestamp
    • Full response text
    • Importance score
    • Metadata (model used, critic score, latency, voice mode, etc.)

╭──────────────────────────────────────────────────────────────────────╮
│  🔍 VIEWING YOUR CONVERSATIONS                                       │
╰──────────────────────────────────────────────────────────────────────╯

  cd ~/Emily1.0

  # View recent interactions
  python scripts/view-interactions.py recent --n 50

  # View only your messages
  python scripts/view-interactions.py recent --role user --n 50

  # View only Emily's responses
  python scripts/view-interactions.py recent --role assistant --n 50

  # Search for specific content
  python scripts/view-interactions.py search "python code"
  python scripts/view-interactions.py search "machine learning"

  # Show statistics
  python scripts/view-interactions.py stats

  # Export everything to JSON
  python scripts/view-interactions.py export-all my_conversations.json

  # Create manual backup
  python scripts/view-interactions.py backup

    save_all_interactions: false
